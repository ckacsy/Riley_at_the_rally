# Project Description

Riley at the Rally is a project that aims to engage users with an immersive experience, providing insights into the world of rallies and automotive sports. Users can explore various aspects of rallying, including driver profiles, car specifications, and event schedules.

# Architecture Overview

The architecture of the project follows a modular approach, which includes:

1. **Frontend**: Built using React, ensuring a responsive and interactive user interface.
2. **Backend**: A RESTful API built with Node.js and Express, facilitating communication between the frontend and the database.
3. **Database**: Utilizes MongoDB for storing user data, event information, and other relevant details.
